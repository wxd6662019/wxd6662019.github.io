算法库

2024/11/18 理论 C++

  1. 头文件include <algorithm>

    算法库中的算法均为非成员函数

  2. 分类

    • 非修改式的算法 for_each
    • 修改式 remove_if
    • 排序
    • 二分搜索
    • 集合操作
    • 堆操作
    • 最值
    • 数值操作
    • 未初始化的内存操作

  3. for_each 算法

    // Apply a function to every element of a sequence. 
for_each(v.begin(), v.end(), [a](int v)mutable
    {
        --a;
        cout << "a=" << a << endl;
        v=v*2;
        cout << v << " ";
    });
//对容器内的所有元素执行此匿名函数
//[] 为捕获列表，默认空表示不捕获， = 表示捕获所有变量（按值）
//& 表示捕获所有变量（按引用）。 mutable 表 你可以修改按值捕获的变量（read-only）的副本

  4. remove_if 算法

    std::vector<int> v{1, 2, 3, 4, 5, 6};

// 移除所有大于 3 的元素
auto it = std::remove_if(v.begin(), v.end(), [](int x) { return x > 3; });
v.erase(it, v.end());
// 注意
//1.不是真正删除：
//std::remove_if 仅重新排列容器中的元素，逻辑上删除的元素仍然存在于容器中，必须使用 erase 方法来真正删除它们。
//2.返回迭代器：
//remove_if 返回一个指向新逻辑结尾的迭代器，后续的 erase 调用应使用该迭代器。
//3.适用容器：
//remove_if 可以与所有支持随机访问迭代器的容器（如 vector、deque 和 array）结合使用，但不适用于 list，因为 remove_if 只能使用随机访问迭代器。
