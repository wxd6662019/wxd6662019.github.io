Container Overview

2024/11/12 理论 C++

标准模板库概述

STL的六大组件

  1. 容器：用来存放数据的，数据结构

    • 序列式容器 vector, list, deque
    • 关联式容器 set, map
    • 无序关联式容器 unordered_set, unordered_map

  2. 迭代器：看成是一种指针，广义指针（具备指针的功能）。可以访问容器中的元素

  3. 算法：操作容器中的元素

  4. 适配器：起到适配的作用

    • 容器适配器 stack，queue, priority_queue

      (容器适配器是对其他容器（如 std::deque、std::list）的封装，它们提供了不同的接口，以实现特定的数据结构行为。适配器不直接存储数据，而是通过组合已有的容器来实现某种数据结构的特性。)

    • 迭代器适配器 front_inserter, back_inserter

      (迭代器适配器可以改变基本迭代器的行为，使其能够适应不同的需求。例如，front_inserter 将元素插入到容器的前端，而普通的迭代器可能只是用于访问或修改元素)

    • 函数适配器

  5. 函数对象（仿函数）：定制化操作

  6. 空间配置器：进行空间的申请与释放操作 （使用+原理+源码）
